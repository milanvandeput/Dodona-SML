### Opdracht
Schrijf een programma dat een 3x3 matrix laat invoeren door de gebruiker.
Het programma berekent de determinant van deze matrix (met de regel van Sarrus).


### Voorbeeld

**Invoer:**

    3
    3
    1 0 0
    0 1 0
    0 0 1

**Uitvoer:**

    1
