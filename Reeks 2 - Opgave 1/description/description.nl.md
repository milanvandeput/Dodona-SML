### Datatypes - opgave 1

We laten de volgende code uitvoeren:

```python
schooljaar1 = "2024-2025"
schooljaar2 = 2024 - 2025

print(schooljaar1)
print(schooljaar2)
```

Wat zal de uitvoer van deze code zijn?

<details>
  <summary>Antwoord:</summary> 
  2024-2025
  \
  -1
